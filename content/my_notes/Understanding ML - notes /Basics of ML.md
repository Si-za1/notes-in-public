the diagram explains a basic way of how it looks like when a model is trained 



features - input
the features can be of various type - qualitative and quantitative if we focus on the most vague terms 
qualitative - nominal data and ordinal data 
quantitative - continuous and discrete data 



model 

output - predictions 
predictions suggests what kind of output is being predicted and how nearer are we to reach the target value 

classification - binary and multiclass
regression - getting the number as the prediction. for example : what might be the temperature today?
reinforcement learning - an agent learning to reach the target value in the best way possible via the feedback mechanism
